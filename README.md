# Englympics-Programming-Team---Wildfire-Game
This project implements an AI agent responsible for controlling a firefighter robot in a real-time simulation. The robot must avoid spreading fires, extinguish flames efficiently, and collect strategic power-ups such as rapid fire, super bullets, and rapid movement boosts.
The challenge included limited decision time per game tick, dynamic enemy behavior, and balancing offense vs. evasion.
